## ЭТАП 1️⃣: ПОДГОТОВКА (10 минут)

### Шаг 1: Проверить и скачать файлы
```
Убедитесь что у вас есть:
✅ index.html
✅ Code.gs
✅ Все файлы документации (MD)
```

### Шаг 2: Получить ID Telegram группы

Способ 1 (Через специального бота):
1. Добавьте бота [@JSonfyBot](https://t.me/JSonfyBot) в вашу группу мастеров
2. Напишите `/info`
3. Скопируйте число после `ID`: -100ХХХХХХХХХХХХ

Способ 2 (Через веб-версию):
1. Откройте группу в браузере web.telegram.org
2. В URL найдите число после `#-`
3. Добавьте спереди `-100`: `-100123456789`

**Запомните этот ID! Он нужен дальше**

### Шаг 3: Получить Google Таблицу для хранения

Новый способ (рекомендуется):

2. Назовите "Заявки клининг"
3. Откройте https://script.google.com/
4. Создайте новый проект
5. Свяжите с таблицей (Tools → Script editor)

## ЭТАП 2️⃣: НАСТРОЙКА GOOGLE APPS SCRIPT (15 минут)

### Шаг 1: Откройте Google Apps Script

1. Откройте Google Таблицу
2. Нажмите **Расширения** → **Apps Script**
3. Откроется редактор кода

### Шаг 2: Замените КОД

1. **Удалите весь старый код:**
   - Выберите всё (Ctrl+A)
   - Нажмите Delete

2. **Вставьте новый код:**
   - Скопируйте весь текст из файла `Code.gs`
   - Вставьте в едитор (Ctrl+V)
   - Нажмите **Сохранить** (Ctrl+S)

3. **Назовите проект:**
   - Нажмите на "Untitled project" сверху
   - Введите "Sistema Zakaza Klininga"
   - Нажмите OK

### Шаг 3: Установите свойства проекта

1. Нажмите иконку **Параметры проекта** (⚙️) слева
2. Перейдите на вкладку **"Свойства"**
3. Добавьте две переменные:

```
Переменная 1:
Ключ: TELEGRAM_BOT_TOKEN
Значение: 8471091759:AAGiszC401WRMWcTXXPi9PizsqxX-oAUurI

Переменная 2:
Ключ: TELEGRAM_CHAT_ID
Значение: -100ХХХХХХХХХХХХ  <-- замените на ваш ID группы
```

4. Нажмите **Сохранить** (Ctrl+S)

### Шаг 4: Проверьте конфигурацию

1. В редакторе нажмите **Execution log** (слева, внизу)
2. Найдите поле для ввода функции (обычно `onOpen()`)
3. Напишите: `__checkConfiguration`
4. Нажмите **Выполнить** (▶️)
5. Посмотрите логи - должны быть ✅ для всех

Ожидаемый результат:
```
=== CONFIGURATION CHECK ===
Bot Token: ✅ Set
Chat ID: ✅ Set (-100...)
Sheet Name: Заявки
========================
```

### Шаг 5: Создайте триггер для напоминаний

1. Нажмите иконку **Часов** (⏰) слева
2. Нажмите **+ Создать триггер**
3. Выберите:
   - Функция: `sendReminders`
   - На основе: "На основе времени"
   - Тип: "Каждый час"
   - Время: "02:00 - 03:00"
4. Нажмите **Сохранить**

Если спросит про разрешение:
- Нажмите на себя
- Выберите "Дополнительно"
- Нажмите "Перейти к проекту"
- Нажмите "Allow"

## ЭТАП 3️⃣: РАЗВЕРТЫВАНИЕ (10 минут)

### Шаг 1: Разверните как веб-приложение

1. Нажмите **Развернуть** (сверху справа)
2. Выберите **Новое развертывание**
3. Выберите тип развертывания: **Веб-приложение**
4. Заполните:
   - **Выполнять как:** Ваша почта (где открыто Apps Script)
   - **Кому предоставить доступ:** все
5. Нажмите **Развернуть**
6. **СКОПИРУЙТЕ URL** из окна "(Copied)"

```
Ожидаемый формат:
https://script.google.com/macros/s/AKfycbz3lOBHg9PTboUskRFWQf8khZD0GkMSXUyF7BkHgZg_sjUIO7sZdbS0kZi47CNAsT8Y/exec
```

### Шаг 2: Обновите URL в HTML

1. Откройте файл `index.html` в редакторе кода
2. Найдите строку с `GOOGLE_SCRIPT_URL` (Ctrl+F → GOOGLE_SCRIPT_URL)
3. Замените URL на **ВАШЕ** значение (из шага 1)
4. Сохраните файл

```javascript
// БЫЛО:
GOOGLE_SCRIPT_URL: 'https://script.google.com/macros/s/AKfycbz3.../exec',

// СТАЛО:
GOOGLE_SCRIPT_URL: 'https://script.google.com/macros/s/ВАШ_URL/exec',
```

### Шаг 3: Откройте HTML в браузере

1. Откройте файл `index.html` в браузере (Ctrl+O или перетащите)
2. ИЛИ загрузите на веб-хостинг и откройте по ссылке
3. Проверьте что загружается главная форма

## ЭТАП 4️⃣: НАСТРОЙКА TELEGRAM (10 минут)

### Шаг 1: Убедитесь что бот в группе

1. Откройте вашу группу мастеров Telegram
2. Проверьте что бот ([@prace_bot](https://t.me/prace_bot) или другой) добавлен
3. Дайте боту права администратора:
   - Нажмите на имя группы
   - Найдите бота
   - Нажмите "Редактировать права"
   - Включите: "Отправка сообщений", "Удаление сообщений"
   - Сохраните

### Шаг 2: Проверьте что бот работает

1. В Google Apps Script запустите:
```javascript
__checkConfiguration()
```

2. Должно вывести что бот готов

3. В консоли Google Apps Script запустите один раз:
```javascript
checkTelegramBotConnection()
```

(Эта функция вызовет getMe на боте)

### ❌ Что делать если не работает

**Ошибка: "Сообщение не отправляется в группу"**
- Проверьте что бот добавлен в группу (**Шаг 4.1**)
- Проверьте что бот имеет права отправлять сообщения
- Запустите `__checkConfiguration()` - смотрите логи

**Ошибка: "При нажатии кнопки ничего не происходит"**
- Проверьте что триггер создан (**Шаг 2.5**)
- Откройте логи Apps Script и смотрите ошибку
- Запустите `__checkConfiguration()`

**Ошибка: "URL неправильный"**
- Скопируйте новый URL из "Развернуть" → "Управление развертываниями"
- Замените в `index.html`
- Перезагрузите страницу браузера

## 🆘 ЕСЛИ ПРОШЛА ОШИБКА

### Диагностика

1. **Откройте Google Apps Script**
2. **Нажмите Ctrl+Enter**
3. **Смотрите логи:**

```
Если видите красное - это ошибка
Скопируйте текст ошибки
```

### Частые ошибки

**"Error: Sheet "Заявки" not found"**
- Решение: Таблица с таким именем не создана автоматически
- Fix: Откройте Google Таблицу, добавьте лист с названием "Заявки"

**"Error: TELEGRAM_BOT_TOKEN not found"**
- Решение: Свойства не установлены
- Fix: Повторите **Этап 2, Шаг 3**

**"Error: Can't delete message"**
- Решение: Бот не имеет прав
- Fix: Дайте боту право "Удаление сообщений"

**"Error: The URL does not exist"**
- Решение: URL неправильный в HTML
- Fix: Скопируйте новый URL и замените
