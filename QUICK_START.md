## 🎯 За 5 минут до первой заявки

### 1. Проверить файлы ✅
```
index.html        ← Обновлен с правильным URL
Code.gs          ← Новый код для Google Apps Script
SETUP_GUIDE.md   ← Полная инструкция (вы сейчас здесь)
MANAGER_GUIDE.md ← Инструкция для менеджеров
MASTER_GUIDE.md  ← Инструкция для мастеров
```

### 2. Обновить Google Apps Script (5 минут)

1. Откройте https://script.google.com
2. Откройте свой проект
3. Откройте файл `Code.gs`
4. **Удалите весь старый код полностью**
5. **Вставьте новый код** из файла `Code.gs`
6. Нажмите **Сохранить** (Ctrl+S)

### 3. Установить свойства (2 минуты)

В Google Apps Script:

**Вариант 1: Автоматически**
```javascript
// Скопируйте и запустите один раз в консоли (Ctrl+Enter):
__setScriptPropertiesForToken()
```

**Вариант 2: Вручную**
- Слева нажмите иконку часов ⚙️
- Перейдите на вкладку "Свойства"
- Добавьте две переменные:
  - Ключ: `TELEGRAM_BOT_TOKEN` | Значение: `8471091759:AAGiszC401WRMWcTXXPi9PizsqxX-oAUurI`
  - Ключ: `TELEGRAM_CHAT_ID` | Значение: `-1003875039787` (замените на ваш ID)

### 4. Создать триггер для напоминаний (1 минута)

1. Слева нажмите иконку часов ⏱️
2. Нажмите **+ Создать триггер**
3. Выберите:
   - Функция: `sendReminders`
   - Событие: "На основе времени"
   - Тип: "Каждый час"
   - Время: "02:00 - 03:00" (или любое)
4. Нажмите **Сохранить**

### 5. Развернуть как веб-приложение (2 минуты)

1. Нажмите **Развернуть** (сверху справа)
2. Выберите **Создать развертывание**
3. Тип развертывания: **Веб-приложение**
4. Параметры:
   - Выполнять как: Ваша почта
   - Кому предоставить доступ: "Все"
5. Нажмите **Развернуть**
6. **Скопируйте URL** из уведомления
7. Проверьте что URL совпадает с URL в `index.html`

## 🚀 Первая заявка

### Менеджер:
1. Откройте `index.html` в браузере
2. Введите ваше имя
3. Заполните форму
4. Нажмите "Создать заявку"

### Что должно произойти:
- ✅ На сайте показалось сообщение "Заявка создана"
- ✅ В Google Таблице `Заявки` появилась новая строка
- ✅ В Telegram группе появилось сообщение с кнопкой "ВЫХОЖУ НА ЗАЯВКУ"

### Мастер:
1. Откройте сообщение в Telegram группе
2. Нажмите кнопку "✅ ВЫХОЖУ НА ЗАЯВКУ"
3. Должно произойти:
   - ✅ Сообщение удалилось из группы
   - ✅ Вы получили полное сообщение в личку с телефоном и адресом
   - ✅ В Google Таблице появилось ваше имя в колонке "Master Name"

## 🆘 Если что-то не работает

### Сообщение не отправляется в Telegram
- Проверьте токен в **Завойства** проекта
- Проверьте что бот добавлен в группу

### Напоминания не приходят
- Проверьте что триггер создан (иконка часов)
- Запустите `__testReminders()` вручную

### URL не работает
- Скопируйте новый URL из "Развернуть" → "Управление"
- Замените в `index.html` в переменной `GOOGLE_SCRIPT_URL`

### Фото не сохраняются
- Убедитесь что отправляешь **фото** а не документ
- Добавьте подпись к фото
