## 🚀 Как настроить систему

### **ШАГ 1: Обновить HTML файл**
✅ **ГОТОВО** - файл `index.html` уже обновлен с правильным URL

### **ШАГ 2: Обновить Google Apps Script**

1. **Откройте Google Apps Script проект:**
   - Перейдите на https://script.google.com
   - Откройте проект для вашей Google Таблицы

2. **Замените весь код в файле `Code.gs`:**
   - Удалите весь старый код
   - Вставьте новый код из файла `Code.gs` (полностью)
   - **Сохраните** (Ctrl+S)

3. **Настройте свойства скрипта:**
   ```javascript
   // В Apps Script консоли запустите эту функцию (Ctrl+Enter):
   __setScriptPropertiesForToken()
   ```
   
   Или вручную:
   - Перейдите в **Проект-параметры** (слева)
   - Добавьте свойства:
     - `TELEGRAM_BOT_TOKEN`: `8471091759:AAGiszC401WRMWcTXXPi9PizsqxX-oAUurI`
     - `TELEGRAM_CHAT_ID`: Ваш ID группы (например: `-1003875039787`)

4. **Установите триггер для напоминаний:**
   - Нажмите на иконку часов (слева)
   - Создайте новый триггер:
     - Выберите функцию: `sendReminders`
     - Тип события: "На основе времени"
     - Тип таймера: "Каждый час"
   - Это будет проверять и отправлять напоминания каждый час

5. **Разверните как веб-приложение:**
   - Нажмите **Развернуть** → **Создать развертывание**
   - Выберите тип: **Веб-приложение**
   - Параметры:
     - Выполнить как: Ваша почта
     - Кому предоставить доступ: "Все"
   - Скопируйте URL развертывания (должен выглядеть как `https://script.google.com/macros/s/AKfycbz3l...`)
   - Замените URL в `index.html` если отличается

### **ШАГ 3: Проверить настройки**

В Google Apps Script запустите проверку:
```javascript
__checkConfiguration()
```

Посмотрите логи (Ctrl+Enter) - должны быть зелёные галочки ✅

### **ШАГ 4: Включить Webhook для Telegram (опционально)**

Для более быстрой обработки можно настроить вебхук:

1. В Google Apps Script добавьте функцию:
```javascript
function setWebhook() {
  const token = PropertiesService.getScriptProperties().getProperty('TELEGRAM_BOT_TOKEN');
  const url = 'https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/usercontent';
  
  const response = UrlFetchApp.fetch(`https://api.telegram.org/bot${token}/setWebhook`, {
    method: 'post',
    payload: JSON.stringify({ url: url }),
    contentType: 'application/json'
  });
  
  Logger.log(response.getContentText());
}
```

2. Запустите эту функцию один раз

## 🔄 Полный процесс работы системы

### **День 1: Менеджер создает заявку**
1. Менеджер заполнит форму на сайте
2. Калькулятор рассчитывает стоимость
3. Менеджер создает заявку кнопкой "Создать заявку"
4. **Что происходит автоматически:**
   - Заявка сохраняется в Google Таблицу
   - **Краткое** сообщение отправляется в Telegram группу города
   - Только одна кнопка: "✅ ВЫХОЖУ НА ЗАЯВКУ"

### **День 2: Мастер видит и берет заявку**
1. Мастер видит краткое сообщение в группе
2. Нажимает кнопку "✅ ВЫХОЖУ НА ЗАЯВКУ"
3. **Что происходит автоматически:**
   - Сообщение удаляется из группы
   - Мастер получает **полное** сообщение в личку с:
     - Полной информацией о заявке
     - Телефоном клиента
     - Полным адресом
     - Инструкциями
   - Google Таблица обновляется:
     - Статус → "Взята"
     - Сохраняется имя мастера
     - Сохраняется время принятия
   - Устанавливаются триггеры напоминаний

### **День 3-4: Напоминания**
- **За 24 часа**: Мастер получает напоминание в личку
- **За 2 часа**: Срочное напоминание в личку

### **День 5: Выполнение и фотографии**
1. Мастер приезжает на объект
2. Отправляет фото в бот (с подписью):
   - Фото прибытия
   - Фото использованной химии
   - Фото до работ
   - Фото в процессе
   - Фото после работ
3. **Фотографии автоматически сохраняются в Google Таблицу**

### **День 6: Завершение**
1. Акт и договор подписаны
2. Оплата проведена
3. Мастер может отправить финальное фото
4. Google Таблица обновляется со статусом "Выполнена"

## 🐛 Решение проблем

### **Проблема: Сообщение не отправляется в Telegram**
- Проверьте `TELEGRAM_BOT_TOKEN` в Script Properties
- Проверьте ID группы в `TELEGRAM_CHAT_ID`
- Убедитесь что бот добавлен в группу с правами администратора

### **Проблема: Напоминания не отправляются**
- Проверьте что триггер создан и включен
- Запустите `sendReminders()` вручную для проверки
- Проверьте логи (Ctrl+Enter)

### **Проблема: Фото не сохраняются**
- Убедитесь что мастер отправляет фото как фото (не документ)
- Проверьте что бот получает сообщение с фото

### **Проблема: URL не совпадает**
- Откройте "Развернуть" → "Управление развертываниями"
- Скопируйте правильный URL
- Замените в `index.html` в переменной `GOOGLE_SCRIPT_URL`
