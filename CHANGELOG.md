- Номер заявки
- Город
- Тип уборки
- Площадь
- Время
- Оплата мастеру

**Сообщение в личку мастеру (полное):**
- Полная информация о заявке
- **Телефон клиента** (в плотное сообщение)
- **Полный адрес** с номером дома и квартиры
- Тип уборки и детали работы
- Оборудование и химия
- Инструкции что делать
- Информация о напоминаниях

#### ✅ Автоматическое удаление сообщения из группы
- Когда мастер нажимает "ВЫХОЖУ НА ЗАЯВКУ"
- Сообщение **сразу удаляется** из группы
- Это предотвращает дублирование и путаницу
- Другие мастера не видят что заявка уже взята

#### ✅ Сохранение информации о мастере в Google Таблицу

**Новые колонки в таблице:**
- Колонка 20: `Master ID` - Telegram ID мастера
- Колонка 21: `Master Name` - Имя и фамилия мастера
- Колонка 22: `Дата принятия` - Когда мастер взял заявку

**При принятии заявки:**
- Статус меняется на "Взята"
- Сохраняется ID мастера (для последующих операций)
- Сохраняется имя мастера (для отчетов)
- Сохраняется точное время принятия

#### ✅ Система напоминаний (24ч и 2ч)

**Триггер:**
- Функция `sendReminders()` запускается каждый час
- Проверяет ВСЕ принятые заявки

**Напоминание за 24 часа:**
- Отправляется за 24ч ± 1ч до работы
- Текст: "Напоминание за 24 часа! Проверьте детали"
- Сохраняется статус в колонку 23

**Напоминание за 2 часа:**
- Отправляется за 2ч ± 30мин до работы
- Текст: "Срочное напоминание! Через 2 часа. Отправьте фото прибытия"
- Сохраняется статус в колонку 24

**Отслеживание:**
- Новые колонки 23-24 в таблице
- Там записываются статусы отправки напоминаний

#### ✅ Архитектура Google Apps Script полностью переделана

**Новые функции:**
- `generateBriefText()` - генерирует краткое сообщение для группы
- `generateFullText()` - генерирует полное сообщение для мастера
- `updateOrderTaken()` - обновляет таблицу когда мастер берет заявку
- `sendReminders()` - отправляет напоминания

**Улучшения:**
- Лучше обработка ошибок
- Лучшее форматирование HTML в Telegram
- Более понятные логи для отладки
- Поддержка фотографий от мастеров

## 🔄 Как обновиться с версии 1.0?

### Шаг 1: Сохразите старую таблицу
```
Копия Google Таблицы (на случай проблем)
```

### Шаг 2: Замените HTML файл
- Сохраните старый `index.html` как `index.html.backup`
- Замените на новый `index.html`
- Проверьте что URL другой

### Шаг 3: Замените Google Apps Script КОД
- Откройте Google Apps Script
- **Удалите весь старый код из Code.gs**
- **Вставьте весть новый код из Code.gs**
- Сохраните

### Шаг 4: Обновите свойства
- Проверьте `TELEGRAM_BOT_TOKEN`
- Проверьте `TELEGRAM_CHAT_ID`
- Запустите `__setScriptPropertiesForToken()`

### Шаг 5: Создайте триггер
- Для старых версий триггер может не быть
- Создайте новый триггер для `sendReminders`

### Шаг 6: Развернуть заново
- Откройте "Развернуть"
- Создайте новое развертывание
- Скопируйте новый URL

### Шаг 7: Проверить
- Запустите `__checkConfiguration()`
- Создайте тестовую заявку
- Нажмите кнопку в группе
