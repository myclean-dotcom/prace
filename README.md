# 🧹 Система управления заявками для клининговой компании

Полнофункциональная система для управления заявками на уборку с интеграцией Telegram и Google Таблицами.

## ✨ Возможности

### 👨‍💼 Для менеджеров:
- 📝 Расчет стоимости уборки онлайн
- 📊 Калькулятор с учетом сложности (1-10 уровней)
- 💰 Автоматический расчет зарплаты мастерам
- 🔧 Расширенная система доп. услуг
- 📱 Отправка заявок в Telegram одним кликом
- 📈 Сохранение всех данных в Google Таблицу

### 🧑‍🔧 Для мастеров:
- 🔔 Краткие уведомления о заявках в группе
- 👥 Одна кнопка для принятия заявки
- 📞 Полная информация с телефоном клиента
- ⏰ Автоматические напоминания (24ч и 2ч)
- 📸 Система отправки фотографий для отслеживания
- ✅ Все данные автоматически сохраняются

### 📊 Для аналитики:
- 📋 Google Таблица с полной историей
- 👤 Отслеживание работы каждого мастера
- 📈 Статистика по заявкам и их статусам
- 💳 Финансовая информация и начисления
- 🖼️ Архив фотографий из работ

## 🌍 Как это работает: Полный цикл

### День 1: Создание заявки
```
Менеджер → Заполнил форму → Создал заявку
                               ↓
[Сайт] → Отправил на Google Apps Script
         ↓
         Сохранил в Google Таблица
         ↓
         Отправил краткое сообщение в Telegram группу
         ↓
         Появилась кнопка "ВЫХОЖУ НА ЗАЯВКУ"
```

### День 2: Мастер берет заявку
```
Мастер → Видит сообщение в группе → Нажал кнопку
                                        ↓
[Telegram] → Удалил сообщение из группы
           ↓
           Отправил полное сообщение в личку мастеру
           ↓
[Google Apps Script] → Обновил Google Таблицу
                      ↓
                      Добавил имя мастера
                      ↓
                      Установил напоминания
```

### День 3-4: Напоминания
```
[Система напоминаний] → Проверяет каждый час
                        ↓
                        За 24ч? → Отправить напоминание
                        ↓
                        За 2ч? → Отправить срочное напоминание
```

### День 5: Выполнение работ
```
Мастер → Приехал на объект → Отправил фото в бот
                                ↓
[Бот] → Получил фото → Сохранил в Google Таблице
                       ↓
                       "Прибытие" → Колонка 27
                       "До работ" → Колонка 28
                       "После работ" → Колонка 29
                       И т.д.
```

## 🔐 Безопасность

- ✅ Не требуется пароль (работает на основе Telegram авторизации)
- ✅ Данные хранятся в Google (безопасно)
- ✅ Телефон клиента видит только назначенный мастер
- ✅ Адрес видит только в личку (не в публичном чате)
- ✅ Все операции логируются в Google Таблице

## 💡 Примеры использования

```
Менеджер:
- Тип: Генеральная
- Площадь: 85 м²
- Сложность: 5 (животные, много мебели)
- Стоимость: 8 500 руб

Мастер получает в группах:
"ЗАЯВКА №CLN-12345678, 85 м², генеральная, 
Новосибирск, 14:00, 5600 руб, 
ул. Ленина 5 (кв. 42)"

Мастер нажимает - получает в личку:
Полный адрес, телефон, время, 
что брать с собой, что делать
```

### Пример 2: Офис после ремонта
```
Менеджер:

- Мастеру: 24 500 руб

Система:
- Сообщение в группу

- Получает полную инфо

- За 2ч напоминание

- Всё сохраняется
```

## 📊 Структура данных

### Google Таблица содержит:
- Номер заявки (CLN-XXXXXXXX)
- Информацию о менеджере
- Данные клиента (имя, телефон, адрес)
- Информацию о работе (тип, размер, сложность)
- Финансовые данные (цена, оплата мастеру)
- Информацию о мастере (имя, ID, когда взял)
- Статус выполнения
- Ссылки на фото
- Время напоминаний

## 🆘 Часто задаваемые вопросы

**В: Нужно ли платить за хостинг?**
О: Нет! Всё бесплатно на Google серверах

**В: Сколько виходов можно делать в день?**
О: Неограниченно! Google Apps Script позволяет миллионы запросов в день

**В: Данные безопасны?**
О: Да! Хранятся в защищенных Google Таблицах с вашей авторизацией

**В: Можно ли добавить другую группу мастеров?**
О: Да! Просто создайте отдельную таблицу для каждого города

**В: Что если мастер не выполнил работу?**
О: Вы можете отметить в таблице или переотправить заявку

**В: Можно ли отредактировать заявку?**
О: Через сайт - нет, но можно в Google Таблице (для менеджеров)
