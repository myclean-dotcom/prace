## 🚀 СЛЕДУЮЩИЕ ШАГИ (В ТОМ ПОРЯДКЕ)

### ШАГ 1️⃣: Обновить Google Apps Script (15 мин)
**Самый важный шаг!**

1. Откройте https://script.google.com
2. Откройте свой проект
3. Откройте файл `Code.gs`
4. **Удалите весь старый код полностью**
5. **Вставьте новый код из файла `Code.gs`** которой лежит у вас в проекте
6. Нажмите **Сохранить** (Ctrl+S)

➡️ **Следуйте инструкции в [QUICK_START.md](QUICK_START.md) бо 5 минут**

---

### ШАГ 2️⃣: Установить свойства (2 мин)
1. В Google Apps Script откройте вкладку Свойства (комбос Project Settings)
2. Добавьте две переменные:
   ```
   TELEGRAM_BOT_TOKEN: 8471091759:AAGiszC401WRMWcTXXPi9PizsqxX-oAUurI
   TELEGRAM_CHAT_ID: -100ХХХХ (замените свой ID группы)
   ```

---

### ШАГ 3️⃣: Создать триггер (1 мин)
1. В Google Apps Script нажмите кнопку "Часов" (⏰)
2. Создайте триггер:
   - Функция: `sendReminders`
   - Тип: "Каждый час"

Это нужно для напоминаний мастерам!

---

### ШАГ 4️⃣: Развернуть как веб-приложение (2 мин)
1. Нажмите **Развернуть** → **Новое развертывание**
2. Тип: **Веб-приложение**
3. Выберите "Все" в доступе
4. **Скопируйте URL**

---

### ШАГ 5️⃣: Обновить URL в HTML (1 мин)
1. Откройте `index.html` в редакторе
2. Найдите строку с `GOOGLE_SCRIPT_URL`
3. **Замените на новый URL** из шага 4
4. Сохраните файл

---

### ШАГ 6️⃣: Проверить что работает (5 мин)
1. Откройте `index.html` в браузере
2. Заполните тестовую заявку
3. Нажмите "Создать заявку"
4. Проверьте что:
   - ✅ Сообщение появилось в Telegram группе
   - ✅ В Google Таблице появилась строка
   - ✅ Кнопка "ВЫХОЖУ НА ЗАЯВКУ" видна

---

### ШАГ 7️⃣ (ОПЦИОНАЛЬНО): Запустить систему полностью

## 🎯 РЕЗУЛЬТАТ

После внедрения система будет работать так:

```
МЕНЕДЖЕР:
1. Открывает сайт
2. Заполняет форму уборки
3. Рассчитывает стоимость
4. Заполняет данные клиента
5. Нажимает "Создать заявку"
   ↓
АВТОМАТИЧЕСКИ:
1. Заявка сохраняется в Google Таблицу
2. Краткое сообщение отправляется в Telegram группу мастеров
   ↓
МАСТЕР:
1. Видит сообщение в группе (только нужная информация)
2. Нажимает кнопку "ВЫХОЖУ НА ЗАЯВКУ"
   ↓
АВТОМАТИЧЕСКИ:
1. Сообщение удаляется из группы (другие не видят)
2. Мастер получает ПОЛНОЕ сообщение в личку с:
   - Телефоном клиента
   - Полным адресом
   - Инструкциями
3. Google Таблица обновляется (имя, ID, время мастера)
4. Устанавливаются триггеры напоминаний
   ↓
ЧЕРЕЗ 24 ЧАСА:
- Мастер получает первое напоминание
   ↓
ЧЕРЕЗ 22 ЧАСА (за 2 часа до):
- Мастер получает срочное напоминание
   ↓
МАСТЕР НА РАБОТЕ:
1. Отправляет фото в бот с подписью
2. Фотографии автоматически сохраняются в Google Таблицу
3. После завершения - заявка закрывается
   ↓
МЕНЕДЖЕР В ТАБЛИЦЕ ВИДИТ:
- Всю историю заявки
- Кто выполнял работу
- Фотографии до и после
- Все финансовые данные
```
