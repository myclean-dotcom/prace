# MANAGER GUIDE

## Первичная настройка через бота
В нужном чате:
- `/setmanager` — сохранить чат менеджера
- `/setevents` — сохранить чат событий
- `/setgroup` — общий чат заявок
- `/setnsk` — чат Новосибирска
- `/myid` — показать текущие идентификаторы

## Рабочие команды
- `/panel` — показать панель кнопок
- `/active` — кто сейчас в работе
- `/planned` — кто назначен на будущие заявки
- `/pay` — интерактивная отправка оплаты

## Как отправить оплату мастеру
Вариант 1:
1. `/pay`
2. выбрать заявку кнопкой
3. отправить текст:
`Ссылка: https://...`
`QR: https://...`

Вариант 2:
`/pay CLN-12345678 https://... QR: https://...`

Вариант 3:
после выбора заявки отправить фото/документ QR — бот перешлет мастеру.

## Кнопка из уведомления
Когда мастер нажал «приехал», менеджеру приходит кнопка:
`💳 Отправить ссылку/QR мастеру`.
Она автоматически переводит бота в режим ожидания оплаты по нужной заявке.
